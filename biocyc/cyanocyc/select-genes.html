Why and how to select a current gene (short)

Click here for longer description

Why select a gene?: For many BioCyc operations you don't have to, but you do if you want to access a specific gene page (perhaps to view its annotation) or to use a specific gene as a focus of an operation.

How to select a gene?

  1. First, you need to have selected the organism that has the gene you're interested in. There's no way of saying "I want to select all isocitrate dehydrogenase genes." You need to start with one in a specific organism. If you don't know how to do this, see this short description or this long description.
     
  2. On many pages, including the organism's front page, you will see a box entitled Search in Current Database. There are several ways you can use this box to indicate the desired gene:
     
    • By Name/ID: Specify any part of the name of the desired gene. For example, if you're looking for alr2339, entering 2339 should be enough. Press Enter. If the name you entered is not specific enough within the selected organism, you'll be presented with a list of possible genes from which to choose. Specifying an NCBI or UniProt ID will also generally work.
       
    • By genetic name: Many genes are known by names used across many organisms. For example, you could find alr2339 within Anabaena/Nostoc PCC 7120 by entering hetR and pressing Enter.
       
    • By function: Entering the function of the protein encoded by the desired will often work. For example, you can enter glu synthetase and get to a list that includes glutamine synthetase (alr2328).
