How to search for orthologs of proteins

What is an ortholog?

  • Orthologs are genes derived from the same ancestral gene, in the last common ancestor of the organisms under consideration [Koonin, 2005].
  • Orthologs often but not always share the same function [Koonin, 2005].
  • Operationally, two proteins are deemed orthologs if each is the best BLAST hit of the other in their respective organisms [Wolf & Koonin, 2012].

How to find orthologs of a protein in different organisms?:

  1. Select a gene. If you don't know how to do this, see this summary or this tutorial.
     
  2. While on the gene page, click Search for this gene in multiple databases in the Comparison Operations menu on the right
     
  3. That will bring up a Find gene in other organisms dialog box. Unclick Same gene name and Same protein name leaving only Ortholog checked. Click OK.
     
  4. From the Specify List of Organism Databases dialog box, choose the target organisms one-by-one by name OR many at once By Taxonomy, by Organism Properties, or by a list. If you don't know how to do this, see this tutorial.
    NOTE: The maximum number of organisms you can search is 70. If a list you select has more than 70 organisms, you'll have to remove them before adding them to Current Selection
  5. Click OK at the bottom right of the Specify List of Organism Databases dialog box to initiate the ortholog search. The time required for the search is roughly proportional to the number of organisms you've selected.
    NOTE: Ortholog searches are done by looking up proteins in a pre-computed table, which is currently incomplete. If an organism does not show an ortholog in the results, it may be that it has no ortholog or that the table of orthologs doesn't yet contain that organism.
